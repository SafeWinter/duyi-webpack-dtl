# S17：Webpack （详细版）学习笔记整理

---



## 1 概况

为吸取上一篇《**S16：零基础 Vue 项目实战**》剥离 `Git` 仓库的教训，实现更好的分支管理和标签管理，渡一前端课程《**Webpack（详细版）**》的所有学习笔记决定单独建立仓库。

课程相关信息：

- 讲师：袁进
- 录制时间：2020 年 1 月
- 所属板块：前端中枢课
- 视频位置：`前端课程/高薪就业阶段/【Webpack详细版】`
- 总体说明：构建工具是工程化的核心内容，对高薪极其关键，但它知识繁杂且深入，需要耗费不少学习时间和精力。此课程旨在更深入地理解构建工具。如果就业时间紧迫，可以先暂时跳过，工作后补上；其余同学 **必须学习**。
