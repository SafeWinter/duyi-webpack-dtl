# L00：学前须知



## 1 课程定位

课程特点：知识点既多且杂，有些地方还有点复杂。

学习前，先要明确学习 `Webpack` 的意义：

:one: 主要是为了方便在今后实际工作中，快速修改基于 `Vue` 或 `React` 等前端框架所搭建的具体项目的 **具体配置细节**；今后的工作大概率是在现成的脚手架上完善改进，很少从头开始搭建脚手架环境。

:two: 深入理解现代前端项目的工程化是如何落地的。



## 2 关于学习方法

相关第三方库的学习：主要是理解工具库的 **设计理念**（如 `CSS` 预编译器、兼容性的处理、优化措施等等）；切忌死记硬背工具库的名称或 `API` 细节。



## 3 关于版本

`Webpack` 这类构建工具属于软件开发的最上层应用，其更新迭代的速度通常也异常迅猛；但其核心理念通常是不变的。学习时严格和视频保持一致即可（`Webpack 4`）。

练手时复制课件的 `package.json`、`package-lock.json` 即可。
